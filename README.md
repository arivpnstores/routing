## Routing Biznet

Untuk mengunduh dan mengatur routing khusus Biznet, jalankan perintah berikut:

```bash
wget -O /root/rules.txt https://raw.githubusercontent.com/arivpnstores/routing/main/rules-biznet.txt
````

## Routing SG

Untuk mengunduh dan mengatur routing khusus Singapore, jalankan perintah berikut:

```bash
wget -O /root/rules.txt https://raw.githubusercontent.com/arivpnstores/routing/main/rules-sg.txt
```

Dengan menggunakan routing di atas, Potato Tunneling akan lebih optimal dalam mengakses layanan sesuai kebutuhan, baik di jaringan Biznet maupun Singapore.
